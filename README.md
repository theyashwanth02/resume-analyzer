# 📄 Resume Analyzer

An AI-powered full stack web application that analyzes resumes and provides detailed feedback instantly.

## 🚀 Live Demo
> Run locally by following the steps below

## ✨ Features
- 📂 Upload PDF resumes
- 🤖 AI-powered analysis using Groq (LLaMA 3.3)
- 📊 Overall score out of 100
- ✅ Strengths & Weaknesses breakdown
- 💡 Actionable improvement tips
- 🎯 ATS Compatibility check

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **AI:** Groq AI (LLaMA 3.3 70B)
- **PDF Processing:** PDF.js

## ⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/yourusername/resume-analyzer.git

2. Install dependencies
   npm install

3. Create a .env file and add your Groq API key
   GROQ_API_KEY=your_groq_api_key_here

4. Run the server
   node server.js

5. Open browser and go to
   http://localhost:3000
